# Visual Odometry

Visual odometry for self driving cars

## How to use.

1. Download the dataset from http://www.cvlibs.net/datasets/kitti/raw_data.php.
2. Unzip it into the "data" directory. 
3. Extract the keypoints using the extract_keypoints.py script.
4. Specify the paths in gui.py script and execute it.

## Notes

1.  In general, at the end of each module is a little script which could be reviewed to understand
the functioning of the code. 
2.  All the parameters are defined in the configs.py file.
